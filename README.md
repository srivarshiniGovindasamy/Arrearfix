# ArrearFix

ArrearFix is a complete full-stack web application designed to aid students in managing their studies, track progress, and optimize learning outcomes through various interactive features.

## Features
1. User Authentication (Login/Register)
2. Dashboard with Pass Probability and Risk Indicators
3. Weakness Analysis
4. PYQ Analyzer
5. Smart Study Planner
6. Crash Mode
7. Backlog Recovery Roadmap
8. Adaptive Mini Test
9. Daily Progress Tracker
10. Smart Alerts
11. Peer Comparison

## Tech Stack
- HTML
- CSS
- JavaScript
- Python (Flask)
- SQLite

## Folder Structure
```
/templates        # HTML files
/static           # CSS and JavaScript files
app.py           # Flask backend
database.py      # Database setup
requirements.txt  # Python dependencies
README.md        # Documentation
```

## Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/srivarshiniGovindasamy/Arrearfix.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Arrearfix
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Set up the database:
   ```bash
   python database.py
   ```
5. Run the application:
   ```bash
   python app.py
   ```

## Usage
Access the application at `http://127.0.0.1:5000` in your web browser after running the app.
